# Trend Radar

Pipeline de dados que monitora tendências da internet automaticamente usando Google Trends.

## Objetivo
Criar um sistema que coleta tendências da internet, processa os dados e gera insights.

## Tecnologias
- Python
- Apache Airflow
- Pandas
- PostgreSQL
